L1 — Data Models (Pydantic v2)

Design principle: DPM is the single handoff artifact between every agent and engine. All source flags ([User provided] / [Derived from X] / [Default — confirm?]) are preserved in model metadata. User can override any derived field before confirming.

SiteRequirements (Section A)

class SiteRequirements(BaseModel):
    site_type:             SiteType
    coverage_area_m2:      float  = Field(gt=0)
    num_floors:            int    = Field(ge=1)
    ceiling_height_m:      float  = Field(default=3.0, ge=2.0, le=15.0)
    wall_type:             WallType
    rf_environment:        RFEnvironment
    regulatory_domain:     str    = Field(min_length=2, max_length=3)
    # Derived — set by engine, overridable by user
    noise_floor_dbm:       Optional[float] = None
    available_airtime_pct: Optional[float] = None
    unii_bands_available:  Optional[list[str]] = None

    @model_validator(mode='after')
    def derive_noise_and_airtime(self):
        lookup = {
            RFEnvironment.OPEN_SPACE:   (-95, 90.0),
            RFEnvironment.OPEN_OFFICE:  (-90, 80.0),
            RFEnvironment.OFFICE:       (-87, 70.0),
            RFEnvironment.DENSE_OFFICE: (-85, 60.0),
            RFEnvironment.INDUSTRIAL:   (-80, 50.0),
        }
        if self.noise_floor_dbm is None:
            self.noise_floor_dbm, _ = lookup[self.rf_environment]
        if self.available_airtime_pct is None:
            _, self.available_airtime_pct = lookup[self.rf_environment]
        return self

ClientDevice (Section B) — key constraint

class ClientDevice(BaseModel):
    device_type:           str
    protocol:              WiFiProtocol
    tx_chains:             int   = Field(ge=1, le=8)
    concurrent_assoc_pct:  float = Field(default=0.75, ge=0.0, le=1.0)
    concurrent_active_pct: float = Field(default=0.50, ge=0.0, le=1.0)

    @model_validator(mode='after')
    def active_le_assoc(self):  # DPM Rule 6: active% ≤ assoc%
        if self.concurrent_active_pct > self.concurrent_assoc_pct:
            raise ValueError("concurrent_active_pct must be ≤ concurrent_assoc_pct")
        return self

    @property
    def active_count(self) -> int:
        return int(self.quantity * self.concurrent_active_pct)

Scale rule: Rate × SS × (channel_width / 20).  Example: MCS 11, 2SS, 80 MHz = 143.4 × 2 × 4 = 1,147 Mbps.  MCS 13, 4SS, 320 MHz = 191.2 × 4 × 16 = 12,237 Mbps.

Strategy Pattern

class LinkBudgetStrategy(ABC):
    @abstractmethod
    def snr_to_mcs(self, protocol: WiFiProtocol, channel_width: int,
                   snr_db: float, spatial_streams: int) -> MCSResult: ...
    @abstractmethod
    def max_mcs_index(self) -> int: ...

class WiFi6LinkBudget(LinkBudgetStrategy):
    def max_mcs_index(self) -> int: return 11   # 1024-QAM 5/6

    def snr_to_mcs(self, ...) -> MCSResult:
        eligible = [m for m in MCS_TABLE.values()
                    if m.snr_required_db <= snr_db and m.mcs_index <= self.max_mcs_index()]
        best      = max(eligible, key=lambda m: m.mcs_index)
        data_rate = best.rate_1ss_20mhz * spatial_streams * (channel_width / 20)
        return MCSResult(data_rate_mbps=data_rate, ...)

class WiFi7LinkBudget(WiFi6LinkBudget):   # inherits WiFi 6 baseline
    def max_mcs_index(self) -> int: return 13   # 4K-QAM 5/6

    def snr_to_mcs(self, protocol, channel_width, snr_db, spatial_streams) -> MCSResult:
        if snr_db >= 36 and protocol == WiFiProtocol.BE:
            eligible = [m for m in MCS12_13.values() if m.snr_required_db <= snr_db]
            if eligible:
                best      = max(eligible, key=lambda m: m.mcs_index)
                data_rate = best.rate_1ss_20mhz * spatial_streams * (channel_width / 20)
                return MCSResult(data_rate_mbps=data_rate, ...)
        return super().snr_to_mcs(protocol, channel_width, snr_db, spatial_streams)

L3 — Airtime Calculator

File: src/engines/airtime.py

Core Formula (Revolution Wi-Fi model)

Airtime per device (%)  =  (App_Throughput_Kbps × TCP_overhead × Aggregation_factor)
                           ÷  Device_Throughput_Kbps  × 100

Available Airtime per Radio (%)  =  RF_Environment_Airtime_%  −  SSID_Beacon_Overhead_%
                                    × OFDMA_efficiency_modifier  (if WiFi 6/7 + small frames)

AP Radios Required  =  ⌈ Σ(Active_Devices × Airtime_per_Device) ÷ Available_Airtime ⌉

VoIP Trap: G.711 (128 Kbps) appears to use 0.64% airtime. Actual: 1.68% — 2.5× due to 160-byte frames (no AMPDU aggregation, high per-frame overhead). Always compute via avg_frame_bytes + latency_sensitive flags.
OFDMA Modifier: When ≥50% of active devices transmit frames <256 bytes, multiply available airtime by 1.30 (30% gain). Matches Rev Wi-Fi guidance. 5G analogy: packing multiple UEs into one slot via scheduler.

Selection Logic

1. Filter by WiFi generation from DPM (WiFi6 → AP3000/AP4000; WiFi7 → AP5020+)
2. Filter by band requirement (6 GHz needed? → AP4000 or AP5020+)
3. Check PoE budget — warn if 802.3bt required but switch not verified
4. If AP5020 on 802.3at: warn TX power reduced on 6 GHz radio
5. If 6 GHz band distribution > 50% + SDR capable: flag dual-6GHz SDR mode option
6. If WiFi 7 + Multi-AP Coord + ap_count ≥ 2: note cluster formation
7. Return: APSelectionResult { model, warnings[], notes[], PoE_class }

CCI Scoring

def cci_score(ap_count: int, available_channels: int) -> tuple[float, str]:
    reuse = ap_count / available_channels if available_channels > 0 else 999
    if   reuse < 1.5:  return reuse / 5.0, "LOW"
    elif reuse < 3.0:  return reuse / 5.0, "MEDIUM"
    else:              return min(reuse / 5.0, 1.0), "HIGH"
# Target: reuse < 1.5 → each channel used by fewer than 1.5 APs on average

L7 — Design Agent + Orchestrator

Design Engine Iterative Loop

def run_design(dpm: DesignParameterManifest, strategy: LinkBudgetStrategy) -> DesignResult:
    for iteration in range(5):  # max 5 iterations
        link_budgets    = run_link_budget(dpm, strategy)           # L2
        airtime         = {band: calculate_airtime(dpm, lb, band)  # L3
                           for band, lb in link_budgets.items()}
        ap_count_cap    = max(a.radios_required for a in airtime.values())
        ap_count_cov    = coverage_driven_count(dpm)               # L2 cell radius
        ap_count_final  = math.ceil(
            max(ap_count_cap, ap_count_cov) * (1 + dpm.network.capacity_growth_pct))

        gap = abs(ap_count_cap - ap_count_cov) / max(1, ap_count_cap)
        if gap <= 0.20: break   # converged — gap < 20%
        dpm = adjust_for_gap(dpm, ap_count_cap, ap_count_cov)     # tune MBR/width

    ap_select    = select_ap(dpm)                                  # L4
    channel_plan = plan_channels(dpm, CapacityPlan(ap_count_final=ap_count_final, ...))  # L5
    validation   = rag_validator.validate(dpm, ...)                # RAG layer
    return DesignResult(link_budgets=link_budgets, capacity_plan=...,
                        channel_plan=channel_plan, ap_selection=ap_select, ...)
